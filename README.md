# logger

> A simple idea and note logging app with search.

![Status: Experiment](https://img.shields.io/badge/Status-Experiment-yellow)

## What It Does

Capture ideas and notes with authentication, organized in a dashboard. Includes fuzzy search powered by Fuse.js for quickly finding logged entries. Tracks user activity and provides a clean interface for brainstorming.

## Tech Stack

`React` `Vite` `TypeScript` `Supabase` `shadcn/ui` `Tailwind CSS` `Fuse.js` `TanStack Query`
